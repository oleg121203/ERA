# ERA Project

## Описание
Базовая структура современного веб-приложения с разделением на frontend и backend части, готовая к масштабированию.

## Структура проекта

### Backend:
- src/
  - controllers/
  - models/
  - routes/
  - services/
  - utils/
  - config/
  - tests/

### Frontend:
- client/
  - src/
    - components/
    - pages/
    - assets/
    - styles/
    - utils/
    - tests/

### Общие файлы:
- docs/
- scripts/
- docker/
